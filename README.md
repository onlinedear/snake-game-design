# 贪吃蛇游戏界面设计

这是一个使用Figma设计的贪吃蛇游戏界面。设计包括游戏区域、控制按钮、分数显示和菜单界面等元素。

## 设计特点

- 经典的贪吃蛇游戏界面，带有现代UI元素
- 响应式设计，适合移动设备和桌面平台
- 清晰的游戏状态显示和控制按钮
- 绿色主题，符合贪吃蛇游戏的传统风格
